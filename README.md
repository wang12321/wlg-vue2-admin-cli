### 单页面脚手架 
采用wlg-cli-temp构建
```
ff2 init <name>
```
建议开发环境版本：
node >= 10
npm >= 6.7

里面集成了公共方法和公共组件

layout里面的代码请不要随意改动！！！

dev:mock 是本地启动项目mock数据

当接口接入之后使用dev即可

命令行添加模板和功能请查看ff2 -h 或 ff2 list

注意：如果启动mock数据 node 版本 不能是12.12.0 -- 12.20.0

使用文档：https://www.processon.com/view/link/615fa8757d9c0806d55a8067#map
